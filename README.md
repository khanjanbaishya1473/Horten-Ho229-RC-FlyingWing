# Horten Ho 229 RC Flying Wing
**IIT Guwahati Aeromodelling Club — Fixed Wing Problem Statement**

## About
Scaled-down 1000mm wingspan RC model of the Horten Ho 229 flying wing.
Built in Autodesk Fusion 360. Airfoil analysis in XFLR5 v6.62.

##  Design Specifications

| Parameter | Value |
|-----------|-------|
| Wingspan | 1000 mm |
| Root Chord | 300 mm |
| Tip Chord | 150 mm |
| Taper Ratio | 0.50 |
| LE Sweep Offset | 100 mm |
| Airfoil | MH 45 (9.85% thickness) |
| Analysis | XFLR5 v6.62 — Re = 150,000 |
| Ribs | 7 per half-span (14 total) — 3mm Aeroply |
| Spars | 2 per half-span — 5×5mm Balsa |
| Skin | 3mm Depron foam |
| Control Surfaces | Elevons — rear 30% chord, outboard 60% span |

---

##  CAD Assembly Structure
- Full rib and spar framework — **not a solid block**
- Ribs exportable as DXF for laser cutting
- Spars slide through pre-cut rib slots
- Depron skin panels bent over rib framework
- Elevons separated at hinge line (70% chord)
- Motor and battery bays designated in center section
